# kzoltany.me

My personal website built with [Next.js](https://nextjs.org/) and hosted on [Vercel](https://vercel.com/).

## Features

- Modern and responsive design
- Clean and minimal codebase
- TypeScript support
- Tailwind CSS for styling
- Next.js for server-side rendering
- Vercel for deployment

## Getting Started

To get started, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/kzolt/kzoltany.me.git
```

2. Install dependencies:

```bash
pnpm install
```

3. Create a `.env` file in the root of the project:

```bash
# Copy from .env.example
cp .env.example .env
```

4. Start the development server:

```bash
pnpm dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Customization

You can customize the website by modifying the following files:

- `src/components/contact-section.tsx` - Contact form
- `src/components/hero-section.tsx` - Hero section
- `src/components/pinned-section.tsx` - Pinned repositories section
- `src/components/projects-section.tsx` - Projects section
- `src/components/skills-section.tsx` - Skills section
- `src/components/ui/field.tsx` - Field components
- `src/components/ui/terminal.tsx` - Terminal components
